# CarsList
Setup Instructions:

1. Place project inside XAMPP htdocs
2. Start Apache + MySQL
3. Import carslist.sql into phpMyAdmin
4. Run frontend:
   cd frontend
   npm install
   npm run dev
5. Open:
   http://localhost:5173

Backend runs at:
http://localhost/CarsList/backend/
